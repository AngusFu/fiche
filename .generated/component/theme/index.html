定制主题

定制主题需要在项目中使用 PostCSS 进行构建。

1. 修改 CSS 变量

在项目的 CSS 目录创建一个 custom.css 文件，根据需要修改以下 CSS 变量：

:root {
  --primary-color: #2196f3;
  --grid-gap: 1em;
}

2. 在你的项目中引入 Clair 的样式

@import url('node_modules/clair/src/css/main.css')
